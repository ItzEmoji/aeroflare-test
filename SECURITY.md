# Security Policy

## Supported Versions
Active support is provided for the latest minor/major versions.

## Reporting a Vulnerability
We prefer vulnerabilities to be reported privately:
1. Navigate to the repository's "Security" tab on GitHub.
2. Click on "Vulnerability disclosure" -> "Report a vulnerability".
3. Provide a detailed description, reproduction steps, and potential impact.

Please report all vulnerabilities exclusively through GitHub's Private Vulnerability Reporting mechanism. Do not open public issues or discussions for security concerns.
