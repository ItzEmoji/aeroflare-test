# Repos-Docs Design Spec

**Topic:** Addition of SECURITY.md, CONTRIBUTING.md, and LICENSE (GPL-3.0) to Aeroflare repository.

## Goal
Establish standard repository documentation and guidelines for security reports, developer contributions, and legal licensing for the Aeroflare Nix binary cache project.

## Document Specifications

### 1. LICENSE
- **Source**: Verbatim GNU General Public License v3.0, retrieved from GitHub's licenses API.
- **Path**: `LICENSE`

### 2. SECURITY.md
- **Content**: Details supported versions and instructions for reporting vulnerabilities privately on GitHub.
- **Path**: `SECURITY.md`

### 3. CONTRIBUTING.md
- **Content**: Onboarding guidelines detailing setup with Nix shell (`nix develop`), lints, tests, formatters, and pull request rules.
- **Path**: `CONTRIBUTING.md`
